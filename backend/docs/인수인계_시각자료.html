7자주 하는 작업 (cookbook)

동 owner가 자주 쓰는 패턴 8개. 전체 cookbook은 ../AGENTS.md §5 참조.

📥 사용자 상태 읽기
티어·관심사·닉네임 등 본질적 사용자 상태 → UserContextService.

📰 다른 동 콘텐츠 읽기

news = await content_reader.get_today_news_for_user(
    uid, top_k=3,
)
📤 이벤트 발행
옆 동에 변경을 알린다. 발행자는 결과를 기다리지 않는다.
from core.event_bus import event_bus
from core.contracts import ConceptMasteredEvent

await event_bus.publish(
    ConceptMasteredEvent(user_id=uid, concept_id=cid),
)
📨 이벤트 구독 (멱등!)
자기 동 __init__.py에서 등록. 같은 event_id 두 번 와도 안전해야 함.
from core.event_bus import event_bus
from core.contracts import ConceptMasteredEvent

async def on_concept(e: ConceptMasteredEvent):
    # event_id UNIQUE 처리 후 부작용
    ...

event_bus.subscribe(ConceptMasteredEvent, on_concept)
🌊 LLM SSE 스트리밍
멘토 채팅·토론. 토큰 단위 스트림. 키 없으면 ExternalServiceError.
from sse_starlette.sse import EventSourceResponse
from core.llm import llm, Message

async def gen():
    async for chunk in llm.chat_stream([Message(...)]):
        yield {"event": "delta", "data": chunk.model_dump_json()}

return EventSourceResponse(gen())
